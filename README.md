# 📊 Análise de Dados com SQL - Vendas

## 🎯 Objetivo
Realizar análise de dados de vendas para identificar padrões de consumo, clientes mais relevantes e desempenho de produtos.

## 🛠 Tecnologias utilizadas
- MySQL
- SQL (DDL, DML, consultas, subconsultas, índices)

## 📁 Estrutura do projeto
- schema.sql: criação do banco de dados
- inserts.sql: inserção de dados
- consultas.sql: análises realizadas

## 🔍 Análises realizadas
- Faturamento total da empresa
- Produtos mais vendidos
- Clientes com maior volume de compras
- Faturamento por período
- Identificação de clientes acima da média de consumo

## 📈 Aprendizados
- Uso de JOINs para integração de dados
- Aplicação de funções agregadas (SUM, AVG)
- Utilização de subconsultas para análise avançada
- Criação de índices para otimização de consultas
